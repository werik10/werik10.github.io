  • Crie projetos Back End (programação Web) complementando os projetos do Front End

HTML5 E CSS3


HTML5 é a versão mais recente da linguagem HTML, usada para estruturar conteúdo na web. Introduz novos elementos semânticos, suporte nativo para multimídia (áudio e vídeo), melhorias em formulários, gráficos 2D com várias novas APIs (como geolocalização e armazenamento local).

CSS3 É a versão mais recente do CSS, usada para estilizar páginas web. Adiciona novas propriedades de estilo (como border-radius e box-shadow), suporte para animações e transições, gradientes, layouts flexíveis e responsivos (Flexbox e Grid Layout), e media queries para design responsivo.

Juntos, HTML5 e CSS3 permitem criar sites modernos, interativos e responsivos.
